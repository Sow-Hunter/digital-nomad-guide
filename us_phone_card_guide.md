## 1. 基础概念

### WiFi Calling（核心功能）

大多数美国电话卡在中国使用都依赖 WiFi Calling。它让你的美国号码通过 WiFi 网络收发短信和接打电话，**资费按美国境内标准计算**，而非天价国际漫游费。

| 支持 WiFi Calling 的手机 | 不支持的手机 |
| --- | --- |
| -   iPhone 6 及以上（国行、港行均可）<br>-   Google Pixel 3 及以上<br>-   部分三星旗舰 | -   华为、一加、OPPO 等（大部分不支持，个别旗舰机可能支持）<br>-   老版 MIUI 的小米手机（HyperOS 已支持，见[第 7 章](#wifi-calling)）<br>-   美版有锁 iPhone |

> [!NOTE]
> 小米 HyperOS 已支持 WiFi Calling
>
> 2025 年后的小米/红米手机（HyperOS 系统）已内置 WiFi Calling 功能。插入美卡后，拨号盘输入 `*#*#869434#*#*` 解除运营商限制，即可开启。详见[第 7 章](#wifi-calling)。
>
> 其他国产安卓（华为、一加等）大部分仍不支持，可通过以下方式使用美国号码：
>
> - **蜂窝漫游接短信**：部分卡（如 Ultra Mobile PayGo）插卡有信号就能收短信，不依赖 WiFi Calling
> - **备用 iPhone**：专门放一台老 iPhone（6s 即可）开 WiFi Calling 接码
> - **eSTK 白卡**：将 eSIM 写入 eSTK，插手机走蜂窝漫游收短信（免费接收）

### 预付费 vs 后付费

本文所有卡都是**预付费（Prepaid）**——先充值后使用，无合约，适合国内用户。后付费卡需要美国地址和信用记录，普通人申请不了。

### T-Mobile vs AT&T 网络

| 维度 | T-Mobile 系 | AT&T 系 |
| --- | --- | --- |
| 中国漫游 | WiFi Calling 支持好，MVNO 选择多 | WiFi Calling 限制多，部分 MVNO 不支持 |
| 号段信誉 | 被号商用得多，部分平台风控 | 号段相对干净 |
| 代表卡 | Tello、Ultra Mobile、1psim、ENC | Red Pocket（AT&T 线路）、Lyca、Cricket |

> [!CAUTION]
> Verizon 网络在中国不可用
>
> Verizon 要求将设备 IMEI 写入系统才能使用。即使激活成功、有信号，也无法正常通话和发短信。**不要买 Verizon 系的卡。**

## 4. Tello Mobile 开卡教程（$5/月）

Tello 是当前最值得个人用户入手的美国实体卡。$5/月包含 WiFi Calling 下 100 分钟通话和无限短信，不开 WiFi Calling 也能在中国蜂窝漫游收发短信（Pay as you go 模式）。

### 独特优势

- WiFi Calling 的 100 分钟可拨打英国、德国等非美国号码（多数便宜套餐只能打美国）
- 官网系统完善，自助管理套餐和充值
- 支持 eSIM
- 不开 WiFi Calling 也能蜂窝漫游（Pay as you go：流量 $10/GB，通话短信按量计费）

### 开卡步骤

1. **准备美国 IP + 美国信用卡**<br>
    Tello 官网对 IP 和支付卡要求严格。需要干净的美国住宅 IP（机场共享 IP 容易被砍单），以及 Visa/Mastercard（可用 Timon 卡、Wise 等；注意 WildCard 已于 2026 年 3 月停服）
2. **访问 tello.com 注册账号**<br>
    填写美国地址（免税州 Oregon / Delaware）
3. **选择套餐**<br>
    Build Your Own Plan → 选 $5/月：100 minutes + Unlimited text + WiFi Calling
4. **选择 SIM 类型**<br>
    推荐选 **eSIM**（即时激活，无需等实体卡邮寄）
5. **支付**<br>
    绑定信用卡付首月 $5。如被砍单，换 IP 或换卡重试
6. **激活 eSIM**<br>
    支付成功后，在 My SIM Section 找到 eSIM 二维码 → 手机扫码安装
7. **开启 WiFi Calling**<br>
    详见[第 7 章](#wifi-calling)

> [!WARNING]
> 砍单率高，准备 Plan B
>
> Tello 没有代理商渠道，只能官网购买。IP 不干净或虚拟卡 BIN 被标记都会被拒。建议准备多个支付方式和不同的代理 IP。如果反复失败，考虑 **Red Pocket**（eBay 购买，流程更简单）或 ENC Mobile 作为替代。

## 5. Ultra Mobile PayGo 开卡教程（$3/月）

> [!CAUTION]
> 2025 年 2 月起大规模封号——新用户慎入
>
> 从 2025 年 2 月起，大量 Ultra Mobile PayGo 用户在**中国激活后 1-2 天内被封号**，表现为无信号、无法登录官网。这是系统性问题，非个例，且封号后账号基本无法恢复。
>
> 如果你还没买，**建议优先选 Red Pocket（$30/年）或 Tello（$5/月）**。如果你已有正常使用中的 Ultra Mobile 号码，可以继续用，但不建议新开卡。
>
> 信源：X 平台多位用户反馈（2025 年 2 月 - 2026 年 6 月持续报告）

曾经是国人用得最多的美国保号卡。$3/月包含 100 分钟 / 100 短信 / 100MB 流量。最大优点是**不开 WiFi Calling 也能蜂窝漫游收短信**，只要有话费余额就行。

### 开卡步骤

1. **购买 SIM 卡**<br>
    淘宝 / 闲鱼搜索「Ultra Mobile PayGo」，也可在亚马逊购买实体卡。eSIM 版本需要在官网操作
2. **准备激活环境**<br>
    - 美国住宅 IP（不是机场共享 IP）
    - 美国地址（可用转运地址）
    - 能付美元的信用卡
3. **访问 ultramobile.com 激活**<br>
    输入 SIM 卡背面的 ACT Code → 选择号码 → 支付 $3 首月费用
4. **插卡使用**<br>
    激活成功后插入手机，等信号出现即可

> [!CAUTION]
> 新卡激活可能被封
>
> Ultra Mobile 风控近年越来越严格。新卡激活有概率被直接封号，官网甚至用代理 IP 都打不开。如果遇到这种情况，换 IP、换卡重试，或改用其他运营商。

> [!TIP]
> Ultra Mobile 的核心优势
>
> **不依赖 WiFi Calling 也能收短信**。在中国插卡有蜂窝漫游信号就能被动接收短信（免费），这是小米等国产安卓用户的福音——不需要 iPhone，不需要 WiFi Calling，插卡就能用。

## 6. Red Pocket 开卡教程（$30/年）

成立于 2006 年的老牌 MVNO，$30/年（$2.5/月）是目前保号成本最低的美国号方案之一。AT&T 号段信誉好，套餐内容丰富（200min / 1000SMS / 200MB）。

### eBay 购买步骤

1. **eBay 搜索 Red Pocket 官方店**<br>
    搜索 `Red Pocket Mobile $30 annual`，找 **Red Pocket Store**（官方店铺，好评率 99.4%，已售 4 万+）
2. **选 SIM 类型**<br>
    **务必选 eSIM**（不是实体卡）。eSIM 无需邮寄，激活码发到邮箱
3. **支付**<br>
    eBay 支持 PayPal / Visa / Mastercard。国内双币卡可用，也可用 Timon 卡
4. **收邮件**<br>
    下单后 Red Pocket 发送 **Confirmation Code** 到 eBay 注册邮箱
5. **激活**<br>
    访问 Red Pocket 官网激活页面 → 输入 Confirmation Code → 选号码 → 获取 eSIM 二维码
6. **写入设备**<br>
    iPhone 扫码添加 eSIM，或用 eSTK 白卡 + 读卡器写入（写卡流程见 [eSIM 写卡教程](giffgaff_esim_guide.md#use-esim)）

> [!NOTE]
> Red Pocket 不支持 WiFi Calling
>
> Red Pocket 的 $30 年费套餐走 AT&T 网络，AT&T 不向 MVNO 开放 WiFi Calling。在中国使用依赖**蜂窝漫游**——手机需有信号才能收短信。接收短信免费，发短信按漫游费计。

> [!WARNING]
> 老用户续费注意
>
> $30/年套餐是 eBay 专享，官网已下架。老用户续费时可能被收 $7 左右的"恢复费"。续费前确认套餐内容未变。

## 7. WiFi Calling 开启指南

WiFi Calling 让美国号码通过 WiFi 网络收发短信和接打电话，资费按美国境内标准。支持 WiFi Calling 的卡包括：Tello、Ultra Mobile、ENC Mobile、1psim。

### iPhone 开启步骤

1. **插入美卡 SIM / 添加 eSIM**
2. **连接 WiFi**（建议用美国 IP 代理，部分运营商首次注册需要）
3. **开启 VoLTE**：设置 → 蜂窝网络 → 点击美卡号码 → 语音与数据 → 选 **4G** 或 **5G**
4. **开启 WiFi 通话**：设置 → App → 电话 → 打开「**Wi-Fi 通话**」
5. **填写紧急地址**：首次开启会要求填写 E911 地址（填美国地址即可）
6. **确认**：状态栏出现 WiFi 标识，或来电显示 "WiFi" 字样

### 小米 / 安卓手机开启步骤

2026 年国内 eSIM 正式开放后，小米 HyperOS 已内置 WiFi Calling 功能，但默认仅对国内运营商开放。插入美卡时需要手动解除运营商限制：

1. **插入美卡 SIM / eSTK 白卡**
2. **解除运营商限制**：拨号盘输入 `*#*#869434#*#*`（HyperOS 适用，解除运营商检查限制），拨出后 WiFi Calling 开关会对外国运营商可用
3. **开启 WiFi Calling**：设置 → SIM 卡与移动网络 → 选择美卡 → 开启「**WLAN 通话**」（或「WiFi 通话」）
4. **开启 VoLTE**：同一页面开启「**VoLTE 高清通话**」
5. **连接 WiFi**（建议美国 IP 代理，部分运营商首次注册 ePDG 需要）
6. **确认**：通知栏或信号栏出现 WiFi Calling 图标

> [!WARNING]
> 不是所有小米/安卓都行
>
> 上述方法在 **HyperOS（2025 年后的小米/红米）**上验证可用。老版 MIUI 或其他国产品牌（华为、一加、OPPO）不一定有效。如果拨号指令无反应或设置中仍无 WiFi Calling 选项：
>
> - 用一台老 iPhone（6s 及以上）专门开 WiFi Calling 接码
> - 选不依赖 WiFi Calling 的卡（Ultra Mobile PayGo 蜂窝漫游可收短信）
> - 刷类原生 ROM（如 Pixel Experience），默认支持 WiFi Calling

### WiFi Calling 常见问题

| 问题 | 解决 |
| --- | --- |
| 开启后无法注册 | 换美国 IP 重试；部分运营商首次需美国 IP 注册 ePDG |
| 状态栏不显示 WiFi 标识 | 重启手机；关闭再开启 WiFi Calling；确认 VoLTE 已开 |
| 能打电话但收不到短信 | 关闭 WiFi Calling 再开；检查手机是否有信号（部分系统优先蜂窝） |
| WiFi Calling 自动断开 | WiFi 网络不稳定或 IP 变化导致 IPsec 隧道断开，重连 WiFi 即可 |

## 8. 号码质量验证

拿到美国号码后，建议先查一下真实属性，确认不是虚拟号（VoIP 号码会被很多平台拒收验证码）。

| 工具 | 网址 | 查什么 | 合格标准 |
| --- | --- | --- | --- |
| FreeCarrierLookup | `freecarrierlookup.com` | 运营商归属 | 显示 T-Mobile / AT&T / 其 MVNO，不是 VoIP |
| Phonevalidator | `phonevalidator.com` | 号码类型 | 显示 `WIRELESS`（非 `VOIP`） |
| IPQualityScore | `ipqualityscore.com` | 风险等级 | 注册账号后 API 查询，结果最准 |

> [!TIP]
> 本文推荐的卡都是真实运营商号码
>
> Tello、Ultra Mobile、Red Pocket、1psim、ENC Mobile 都是正规 MVNO，号码是真实无线号码（WIRELESS），不是 VoIP。不会被 ChatGPT、TikTok、Telegram 等平台拒收验证码。

## 9. 避坑与注意事项

### 通用注意事项

| 坑 | 说明 | 应对 |
| --- | --- | --- |
| **激活 IP** | 多数美卡官网要求干净的美国 IP，机场共享 IP 大概率被拦截 | 用住宅代理或独享 IP；WARP 可尝试但不保证 |
| **信用卡风控** | 虚拟卡 BIN 被标记、地址不匹配会砍单 | 用 Timon 卡（尼区 BIN 较新）或 Wise 卡 |
| **自动续费** | 部分套餐默认开启自动续费，扣完余额后可能欠费停机 | 激活后立即检查并关闭不需要的自动续费 |
| **保号规则** | 长期不使用可能被回收号码 | 设日历提醒，定期发短信或充值保号 |
| **携号转出** | 小运营商可能限制号码转出 | 入手前确认是否支持 port-out |

### 保号周期速查

| 运营商 | 保号条件 | 回收周期 |
| --- | --- | --- |
| Red Pocket | 年费到期前续费 | 到期未续即停 |
| Ultra Mobile | 每 30 天充值 $3 | 余额耗尽后约 60 天 |
| Tello | 每月自动扣费 | 欠费后约 30 天 |
| 1psim | 每月充值 $3 | 欠费后约 30 天 |
| ENC Mobile | 每月充值 $2.99 | 欠费后约 30 天 |

### iPhone 插卡防扣费 Checklist

美卡插入 iPhone 时，系统可能自动触发 iMessage / FaceTime 激活短信，产生国际短信费。插卡前：

1. **连接 WiFi**
2. **关闭 iMessage**：设置 → 信息 → 关闭 iMessage
3. **关闭 FaceTime**：设置 → FaceTime → 关闭
4. **插卡**，弹窗全部点**取消/拒绝**
5. **默认数据卡指向国内主卡**
6. **关闭美卡的数据漫游**
