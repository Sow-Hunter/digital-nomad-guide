eSIM 工具卡写号教程

为什么需要 eSIM 工具卡

Timon 钱包登录和重要操作需要接收尼日利亚手机号的短信验证码。但你人在国内，没有尼日利亚实体 SIM 卡。商家提供的是一个 eSIM 配置文件（二维码形式），需要写入手机才能接码。

eSIM 是什么
eSIM 不是实体卡，而是一段数字配置文件（包含手机号、运营商信息等）。正常情况下写入手机内置芯片即可，但国行手机禁用了 eSIM 功能。eSIM 工具卡就是一张可编程的实体 nano-SIM 卡，把 eSIM 配置写进去，插到手机卡槽里当普通 SIM 卡用。
海外手机的正常流程： eSIM 配置文件 → 写入手机内置芯片 → 直接识别号码 国行手机的流程（用 eUICC 白卡）： eSIM 配置文件 → 读卡器 + 电脑写入 eSTK 白卡 → 插到手机卡槽 → 当普通 SIM 卡识别

eSIM 白卡（eUICC 卡）产品对比

市面上主流的 eSIM 白卡都内置 eUICC 芯片（Embedded Universal Integrated Circuit Card），符合 GSMA 标准，MTN 的 eSIM 配置文件可以写入任何品牌的白卡。

产品价格存储容量可存号码写卡方式开源特点
eSTK ~¥24 840KB ~40 个 读卡器 + 电脑/手机 是 性价比最高，需配读卡器（¥20-30）
9eSIM V0 ~¥85 中等 入门 读卡器 + 电脑/手机 是 入门版
9eSIM V3 ~¥133 1.6MB ~50 个 读卡器 + 电脑/手机 是 容量最大，适合管理大量号码
为什么价格差异大
  • 芯片存储容量：9eSIM V3（1.6MB）是 eSTK（840KB）的近 2 倍，能存更多号码
  • 写卡方式：eSTK/9eSIM 用读卡器 + 开源软件写卡，数据存本地，不依赖任何厂商服务器
  • 5ber 已停运：曾经最流行的 5ber 系列已跑路/停止国内销售，国内无法购买。eSTK 和 9eSIM 是当前主流替代品
推荐：eSTK + 读卡器（总 ~¥49）
你只需要 1 个 MTN 号接验证码，eSTK 的 840KB 存储绰绰有余。开源 + 数据存本地 + 不依赖厂商 = 长期无后顾之忧。

写卡方式说明

白卡写卡有两种方式，根据你的手机兼容性选择：

方式原理适用品牌工具
读卡器 + 电脑写卡（推荐） 卡插入 USB 读卡器接电脑，用电脑软件写入，完全绕过手机兼容性问题 eSTK、9eSIM MiniLPA（macOS）/ EasyLPAC（Windows）
手机 App 直写 卡插入手机卡槽，App 通过 OMAPI 接口和卡通信（需手机兼容） 部分品牌（如已停运的 5ber） EasyEUICC（开源）
小米 15 兼容性问题

小米 15 的 OMAPI 接口受限，EasyEUICC 等手机端写卡 App 直接插卡写入不兼容（提示设备不兼容）。这是国行安卓厂商的普遍问题，不是白卡本身的问题。

解决方案：用读卡器 + 电脑写卡，完全绕过手机兼容性限制。写完后拔出来插到小米 15 正常使用，日常接短信不受影响。

eSTK + 读卡器写号步骤（推荐）

  1. 购买 eSTK 白卡 + USB-C 读卡器：淘宝/京东搜"eSTK eSIM"（~¥24）+ "eSIM 读卡器"（~¥25），总计约 ¥49
  2. 电脑安装写卡软件：
    • macOS：下载 MiniLPA
    • Windows：下载 EasyLPAC
  3. eSTK 卡插入读卡器，读卡器 USB 接电脑
  4. 打开写卡软件：软件识别到读卡器和卡片后，显示卡片信息和剩余空间
  5. 写入 eSIM：将商家给的 MTN eSIM 二维码图片拖拽到软件窗口（或扫码），点确认开始写入
  6. 启用号码：写入完成后，右键点击卡片 → 启用（Enable）
  7. 拔出卡，插入小米手机：等待几分钟，信号栏出现 MTN 标识 → 成功

多号码管理（进阶）

如果以后往卡里写入更多号码（如美国号、其他国家号），同一时间只能激活 1 个号码，切换方式：

场景操作说明
电脑切换读卡器接电脑 → MiniLPA 禁用当前号 → 启用目标号最可靠
安卓手机切换EasyEUICC App（需手机 OMAPI 兼容）小米 15 不兼容，需用电脑
STK 菜单切换设置 → SIM 卡应用程序 → eSTK 菜单部分手机支持

* "下载次数"只消耗在写入新号码时。在已写入的号码之间切换不消耗次数，无限次。删除号码也不消耗次数。eSTK 下载次数无限。

MTN 是什么
MTN Nigeria 是尼日利亚最大的电信运营商（隶属南非 MTN Group），相当于国内的中国移动。覆盖最广、信号最稳，是商家默认选择的运营商。

写入完成后

  • 白卡现在就是一张永久的尼日利亚 MTN 实体 SIM 卡，和普通 SIM 卡使用方式完全相同
  • 插进手机就能收短信（Timon 验证码），不需要任何 App
  • 不需要网络连接，短信走运营商蜂窝信号
  • 换手机也能用 — 拔出来插到另一台手机就行
  • 只需要接短信，不需要打电话或流量，不用买任何套餐

MTN 号码日常费用与保号

MTN 是预付费（Prepaid）模式，没有月租。不打电话、不发短信、不用流量就不产生费用。接收短信（Timon 验证码）免费。

保号防回收

尼日利亚通信委员会 (NCC) 规定，SIM 卡连续 6 个月没有任何收入活动（充值、通话、发短信、买流量），运营商有权回收号码。MTN 提供专门的保号套餐：
